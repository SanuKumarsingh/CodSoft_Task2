# 🧮 Calculator — CodSoft Task 2

A simple command-line **Calculator** built in Python as part of the **CodSoft Python Programming Internship (Task 2)**. Performs basic arithmetic operations based on user input using Python's `match-case` statement.

## 🔗 Repository

[github.com/SanuKumarsingh/CodSoft_Task2](https://github.com/SanuKumarsingh/CodSoft_Task2)

## ✨ Features

- Supports **addition, subtraction, multiplication, and division**
- Clean input-based interface — enter two numbers and an operator
- Uses Python's modern `match-case` (structural pattern matching)
- Handles invalid operators gracefully

## 📁 Project Structure

```
CodSoft_Task2/
└── Calculator.py   # Main Python script
```

## 🚀 Getting Started

**Requirements:** Python 3.10+ (for `match-case` support)

```bash
git clone https://github.com/SanuKumarsingh/CodSoft_Task2.git
cd CodSoft_Task2
python Calculator.py
```

## 🖥️ Sample Output

```
Enter number 1: 10
Enter number 2: 5
Enter Operator: *
Product of these no. will be :  50
```

| Operator | Operation      |
|----------|----------------|
